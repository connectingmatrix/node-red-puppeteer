Description

Connect to Maya's browser instance

Inputs

connectionType enum
If value is "new", it will start a new browser session on your local machine. If value is "existing", it will connect to a browser thats already running (this browser could also be running remotely)
showBrowser enum
If set to "yes", Maya will open the browser in headful mode (i.e., in the foreground -this will take considerably more system resources). Otherwise, the browser will run in headless mode (i.e. in the background)
WS Link enum
If connectionType is "existing", this field lets you specify the link to the browser you want to connect to.

Outputs

This node does not have any outputs. If the connect operation is successful, it simply forwards the message.
