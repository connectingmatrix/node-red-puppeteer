Description

Open a page in a new tab

Inputs

URL string
The URL of the page to open
waitUntil enum
How long the node should wait before forwarding the message to the next node. What these options mean -
  • load - consider navigation to be finished when the load event is fired.
  • domcontentloaded - consider navigation to be finished when the DOMContentLoaded event is fired.
  • networkidle0 - consider navigation to be finished when there are no more than 0 network connections for at least 500 ms.
  • networkidle2 - consider navigation to be finished when there are no more than 2 network connections for at least 500 ms.
viewport enum
If set to "default", the browser will open the page with dimensions 1600x900. If set to "custom", you can specify your own viewport.

Outputs

pageIds array[string]
Array that contains the newly opened page's pageId as the first element. This array contains only one element.
