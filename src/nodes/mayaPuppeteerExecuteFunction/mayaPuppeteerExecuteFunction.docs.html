Description

Execute a method defined on an element on the page

Inputs

Page ID string
The pageId of the page that contains the target element
Xpath string
The xpath of the target element
Timeout int
Time (in milliseconds) for which the node will wait for the target element ( as specified by the xpath) to appear on the page, if it isn't already there
Element index int
If multiple elements correspond to the given xpath, this is the index of the desired element
Function name string
Name of the target element's method that needs to be executed. For example, if the target element is a video tag, the function name could be "play"
Arguments json array
List of arguments - in order - that will be passed to the function

Outputs

This node does not have any outputs. If the execute-function operation is successful, it simply forwards the message.
