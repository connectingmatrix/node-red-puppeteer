Description

Upload a file on a page

Inputs

Page ID string
The pageId of the page that contains the target input field
Input Xpath string
The xpath of the target input field
Index int
If multiple elements correspond to the given xpath, this is the index of the desired element
Timeout int
Time (in milliseconds) for which the node will wait for the target input field ( as specified by the xpath) to appear on the page, if it isn't already there
File path string
Absolute path of the file to be uploaded

Outputs

This node does not have any outputs. If the input operation is successful, it simply forwards the message.
