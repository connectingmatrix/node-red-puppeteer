Description

Perform a keyboard action (typing, key-press, key-hold) on a page

Inputs

Page ID string
The pageId of the page that the action is to be performed on
actionType enum
The keyboard action you want to perform. This can be "type", "keypress", "keydown", "keyup". They mean the following -
  • type - type some text into an input
  • keypress - simulate pressing a key on the page
  • keydown - simulate holding down a key on the page
  • keyup - simulate releasing a key on the page

If action is "type"

  • Text - The text to type
  • Keypress delay - the delay to simulate between each key-press during typing
  • Xpath - the xpath of the input field to type into
  • Timeout Time (in milliseconds) for which the node will wait for the target input field (as specified by the xpath) to appear on the page, if it isn't already there
  • Element index - If multiple elements correspond to the given xpath, this is the index of the desired element

If action is "keypress"

  • Key name - the name of the key to be pressed. You can get the key name of any key here
  • Delay - the delay (in milliseconds) between pressing the key and releasing it

If action is "keyup" or "keydown"

  • Key Name - the name of the key to be pressed. You can get the key name of any key here

Outputs

This node does not have any outputs. If the input operation is successful, it simply forwards the message.
