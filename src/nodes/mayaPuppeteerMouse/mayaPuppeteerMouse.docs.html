Description

Perform a mouse action (click, hover, scroll) on a page

Inputs

Page ID string
The pageId of the page that the action is to be performed on
actionType enum
The keyboard action you want to perform. This can be "click", "hover", "scroll" They mean the following -
  • click - click an element
  • hover - hover over an element
  • scroll - scroll a specific element or the entire page

If action is "click" or "hover

-
  • Xpath - the xpath of the input field to click or hover over
  • Timeout Time (in milliseconds) for which the node will wait for the target element (as specified by the xpath) to appear on the page, if it isn't already there
  • Element index - If multiple elements correspond to the given xpath, this is the index of the desired element

If action is "scroll"

-
  • scrollType - kind of scroll behaviour required. If value is "intoView", the page will scroll till target element is visible. If value is "byAmount", the page (or element) will scroll by user-specified amount. You can specify a huge amount to scroll to the bottom of the page.
  • Xpath - xpath of the element to scroll or to scroll-into-view
  • Index - If given xpath corresponds to multiple elements, this index specifies the target element among those

Outputs

This node does not have any outputs. If the mouse operation is successful, it simply forwards the message.
