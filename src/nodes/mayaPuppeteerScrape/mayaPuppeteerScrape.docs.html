Description

Scrape data from a tab (without necessarily switching to it) using Maya WebQL


Config

Page ID string
The pageId of the page from which you want to scrape
query JSON
The Maya WebQL query to execute on the page. Refer to WebQL Documentation to learn more.
Timeout number
The maximum amount of time that Maya will wait for each xpath in the query to resolve

Outputs

The following properties are set on the msg object in case of successful execution.

result json
The result of the scraping operation.
